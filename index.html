    Getting Started

    First download and install LOVE2D from this website, then download the .zip of this repo and unpack it. Start the game by either dragging the unpacked folder onto love.exe or by opening up the console and typing the command: "C:\Program Files\LOVE\love.exe" "C:\games\mygame" (commands may vary).

    For more information on LOVE2D visit: https://love2d.org/
